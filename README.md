# quick_linux_environment

Turn a fresh ArchLinux with i3 environment into a beautiful and modern Linux

# Requirements
- ArchLinux

- i3-wm

- git

# Installation
## 1. Clone this repository
```
git clone https://github.com/phangelim/quick_linux_environmet
```

## 2. Enter in the directory of the project
```bash
cd quick_linux_environmet
```

## 3. Run the boot.sh
```
chmod +x boot.sh && ./boot.sh
```
